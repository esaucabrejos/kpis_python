# KPIs Python

Script en Python para el cálculo y análisis de indicadores clave de desempeño (KPIs).

## Descripción
Este proyecto tiene como objetivo automatizar el cálculo de KPIs a partir de datos de entrada, facilitando el análisis y la toma de decisiones.

## Funcionalidades
- Cálculo de KPIs básicos
- Procesamiento de datos mediante Python
- Salida de resultados por consola

## Requisitos
- Python 3.x

## Ejecución
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/esaucabrejos/kpis_python.git
   ```
2. Entrar al directorio del proyecto:
   ```bash
   cd kpis_python
   ```
3. Ejecutar el script:
   ```bash
   python kpis.py
   ```
